### API 接口

| service_code | 含义                                  |
| ------------ | ------------------------------------- |
| **2000**     | 操作成功                              |
| **4001**     | 参数不合法（缺少字段、格式错误）      |
| **4004**     | 数据不存在（任务不存在）              |
| **5001**     | 系统内部错误                          |
| **5002**     | 外部依赖失败（比如 Redis/MySQL 错误） |
| **5003**     | LLM 服务不可用                        |
